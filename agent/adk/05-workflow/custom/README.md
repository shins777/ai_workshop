# ADK 커스텀 워크플로우 (05-workflow/custom)

이 예제는 기본 제공되는 순차/병렬/루프 에이전트 외에, 개발자가 직접 에이전트 간의 상호작용 로직을 세밀하게 제어하는 **Custom Workflow** 패턴을 보여줍니다.

## 주요 구성 요소

### 1. 에이전트 정의 (`agent.py`)
- **커스텀 오케스트레이션**: 루트 에이전트가 다른 에이전트나 도구를 단순히 호출하는 것을 넘어, 특정 조건에 따라 동적으로 워크플로우를 변경하거나 데이터를 가공하는 방식을 커스터마이징할 수 있습니다.

### 2. 비평가 에이전트 (`critic.py`)
- **전용 검증 로직**: 결과물의 품질을 특정 기준에 따라 평가하고, 점수를 매기거나 피드백을 생성하는 독립적인 비평 로직이 포함되어 있습니다.

### 3. 하위 에이전트 (`sub_agent.py`)
- 업무를 실제로 수행하는 Worker 에이전트들로 구성됩니다.

## 특징
- **유연성**: ADK에서 제공하는 표준 워크플로우 패턴으로 해결하기 어려운 복잡한 비즈니스 로직이나 다단계 검증 프로세스를 구현할 때 적합합니다.
- **세부 제어**: 에이전트 간에 전달되는 메시지나 상태(State)를 직접 조작할 수 있는 예제를 제공합니다.

## 실행 방법
`05-workflow` 폴더에서 `adk web`을 실행한 후, 에이전트 목록에서 `custom`을 선택하여 테스트할 수 있습니다.
